Ember-Bootstrap-Modal
=====================

This is a simple Ember.js modal built on Todo app from http://todomvc.com/.

Libraries: 

* Ember.js v1.0.0
* Ember Data v1.0.0-beta2
* Bootstrap v2.3.2
